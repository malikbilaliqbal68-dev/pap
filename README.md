# Paperify - Educational Paper Generator

## 🎯 What is Paperify?

An educational platform for Pakistani students and teachers to generate custom exam papers with MCQs, short questions, and long questions. Supports Punjab, Sindh, and Federal boards with bilingual content (English/Urdu).

---

## ✨ Features

### 1. **Custom Paper Generation**
- Select board (Punjab/Sindh/Federal)
- Choose class (9th, 10th, 11th, 12th)
- Pick subjects and chapters
- Configure question types and counts
- Generate bilingual papers (English/Urdu)
- Download as PDF
- Print-ready format

### 2. **Interactive Quiz System**
- MCQ-based quizzes
- Instant feedback
- Score tracking
- Wrong answer review
- Bilingual questions
- Weak area analytics

### 3. **AI Career Mentor**
- Personalized career roadmaps
- Free course recommendations (YouTube-first)
- Weekly learning plans
- Project submission & review
- Free hosting for projects
- Laptop buying guides

### 4. **User Management**
- Secure registration & login
- Password encryption (bcrypt)
- Session management
- User profiles
- Subscription tracking

### 5. **Payment System**
- Manual payment verification
- JazzCash/EasyPaisa integration
- Multiple subscription plans
- Admin approval system
- Payment screenshot upload
- Transaction ID tracking

### 6. **Referral System**
- Unique referral codes
- Refer 10 paid users = Lifetime free access
- Track referral progress
- Automatic rewards

### 7. **Collaboration Features**
- Share papers with staff
- Collaborative editing
- Paper history
- Save & reload papers

### 8. **Analytics**
- Weak area tracking
- Performance insights
- Quiz statistics
- Progress monitoring

- **Database:** SQLite

- **Authentication:** bcrypt, express-session
- **File Upload:** Multer
- **Payment:** Manual verification
